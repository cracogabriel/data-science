Para baixar os datasets:

1. Acesse o Kaggle (é necessário ter conta): https://www.kaggle.com/datasets/fmpugliese/steam-all-games-data
2. Faça login e abra a aba "Data".
3. Baixe os 3 arquivos .csv disponíveis (all_data.csv, id_name.csv e steam_app_data.csv).
4. Extraia os arquivos e mova-os para a pasta do projeto (data/).
